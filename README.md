# On Javascript

## Usage

evaluate literate javascript using [node](http://nodejs.org/):

    litjs script.js.md

evaluate literate javascript using [rhino](https://developer.mozilla.org/de/docs/Rhino):

    litjs -e rhino script.js.md

use a watcher with a `-w` option

# Install

    brew install node
    brew install rhino

Install `lijs`:

    make
